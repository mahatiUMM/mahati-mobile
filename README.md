# Mahati Mobile

## Introduction

Mahati is Mobile Aplikasi Sahabat Hipertensi. Created at January 2024.

## How to Run

- Clone Project
- Install depedencies on [pubspec.yaml](./pubspec.yaml)
- Run and Debug on android emulator
